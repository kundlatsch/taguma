Bem-vindo ao **Manual de Backend**, o material de apoio para o curso de backend do YouTube.

Este manual cobre os fundamentos e conceitos essenciais para desenvolver aplicações backend robustas, desde os protocolos de rede até a arquitetura de sistemas.

## O que você vai aprender

- Protocolo HTTP e princípios REST
- Bancos de dados relacionais e não-relacionais
- Autenticação e autorização
- Deploy e infraestrutura
