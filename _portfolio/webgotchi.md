# 🧩 Webgotchi

Webgotchi é minha implementação web dos famosos tamagotchis, ou “bichinhos virtuais”. A proposta do projeto foi desenvolver um sistema leve e direto ao ponto, sem uso de frameworks e sem backend, pensado para rodar exclusivamente no GitHub Pages.

Todo o estado do jogo é serializado e salvo como um hash no `localStorage` do navegador. Além disso, o progresso pode ser exportado em arquivos `.tama`, permitindo persistência manual e compartilhamento do estado do jogo. O projeto inclui funcionalidades clássicas de um tamagotchi, como alimentação, higiene, saúde e minigames.

É um projeto adequado para primeiras contribuições, especialmente para quem quer entender o fluxo básico de colaboração em projetos open source. Basta clonar o repositório, criar uma feature branch, implementar as modificações e abrir um pull request. Mais detalhes estão disponíveis no README do GitHub.

Cuidado para não matar seu bichinho sem querer.
